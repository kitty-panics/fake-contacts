# Fake Contacts

批量生成用于测试的假联系人数据。

## 使用

1. `git clone --depth=1 git@github.com:kitty-panics/fake-contacts.git`
2. `cd fake-contacts/bin`
3. 生成联系人数据
    + 生成 csv 格式的数据
        + `./fake-contacts csv 数量 > Contacts.csv`
    + 生成 vcf 格式的数据
        + `./fake-contacts vcf 数量 > Contacts.vcf`
